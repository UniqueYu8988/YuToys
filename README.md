# 🏆 YuToys - 极致视觉桌面助手

![YuToys Badge](https://img.shields.io/badge/YuToys-v1.0.0-blueviolet?style=for-the-badge)

### 📈 量化您的每一分进步 (Statistics)

**数据看板**：忠实记录您的成长足迹。通过可视化展示“达成目标”、“专注时长”、“累计饮水”与“运行时间”四个核心维度，让您的努力有迹可循。YuToys 的数据系统不只是数字的堆砌，更是对您健康生活与高效工作的深层致敬。

### ⚙️ 精细至毫末的控制 (Settings)

**个性化中心**：麻雀虽小五脏俱全。

- **窗口置顶**：让专注时刻始终处于视野焦位。
- **品牌逻辑**：全量控制开机自启与本地化精选铃音。
- **重置机制**：支持二阶段确认的物理级数据清空，一切尽在掌控。

## ✨ 核心特性

- 🎨 **极致美学**：采用 Glassmorphism（玻璃拟态）设计，支持透明磨砂质感与沉浸式动效。
- 🔔 **品牌化通知**：支持整点报时、饮水提醒，所有音效均已本地化，支持 100% 离线运行。
- 🚀 **性能卓越**：采用 Electron + Vite 架构，经过 v16 版本的极致启动优化（无压缩秒开技术），实现秒级响应。
- 💾 **数据持久化**：自动保存任务清单、累计饮水量及窗口位置信息。

## 🛠️ 技术栈

- **前端**: React 19 + TypeScript
- **状态管理**: Zustand
- **动效**: Framer Motion + Canvas Confetti
- **核心框架**: Electron 40
- **构建工具**: Vite + Electron Builder

## 📦 如何运行

如果您是开发者，可以尝试自行构建：

1. **安装依赖**
   ```bash
   npm install
   ```
2. **开发模式运行**
   ```bash
   npm run dev
   ```
3. **分发打包**
   ```bash
   npm run build
   ```
