## 🛡️ 专业级打磨 (Technical Polish)

- ⚡ **秒级冷启动**：采用 `store` 级零损解压技术。
- 🔔 **品牌化通知**：精准适配 Windows 原生通知中心。
- 🔊 **100% 离线运行**：所有音效与动效资源离线可用。
- 💾 **环境自适应**：数据全量同步，永不丢失。

---

## 🛠️ 技术栈 (Tech Stack)

- **前端框架**: React 19 + TypeScript
- **状态管理**: Zustand
- **动效引擎**: Framer Motion + Canvas Confetti
- **底层架构**: Electron 40 + Vite
