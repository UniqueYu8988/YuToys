### ⏳ 深度专注空间

> **见证时间的消融**。支持 1-99 分钟自定义番茄钟，紫色圆环随倒计时优雅消逝。单击圆盘即可随时暂停或恢复。计时结束不仅有精准提醒，更有**持久且震撼的烟花奖励**。

## 🛡️ 专业级打磨 (Technical Polish)

- ⚡ **秒级冷启动**：采用 `store` 级零损解压技术，实现极致“秒开”。
- 🔔 **品牌化通知**：结合 Windows 原生通知，实现整点报时与品牌标识统一。
- 🔊 **100% 离线运行**：所有铃声与动效资源均已内置，脱离网络依赖。
- 💾 **环境自适应**：自动同步开发环境与打包版数据，数据永不丢失。

---

## 🛠️ 技术栈 (Tech Stack)

- **前端平台**: React 19 + TypeScript
- **交互引擎**: Framer Motion + Canvas Confetti
- **状态流转**: Zustand
- **桌面环境**: Electron 40 + Vite
