## 🛡️ 专业级打磨 (Technical Polish)

- ⚡ **秒级冷启动**：采用 `store` 级零损解压技术，便携版亦能实现“秒开”体验。
- 🔔 **品牌化通知**：配置 `AppUserModelId`，通知弹窗正式更名为 “YuToys” 并挂载专属图标。
- 🔊 **100% 离线运行**：所有铃声与动效资源均本地化，无网络亦能享受完整服务。
- 💾 **环境自适应**：自动同步开发环境与打包版数据，数据跟随，绝不丢失。

---

## 🛠️ 技术栈 (Tech Stack)

- **前端平台**: React 19 + TypeScript
- **交互引擎**: Framer Motion + Canvas Confetti
- **状态流转**: Zustand
- **桌面环境**: Electron 40 + Vite

## 📦 开发者指南

```bash
# 1. 注入灵魂 (安装依赖)
npm install

# 2. 开发者预览 (运行)
npm run dev

# 3. 锻造成品 (构建)
npm run build
```
