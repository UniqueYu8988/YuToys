# 🏆 YuToys - 极致视觉桌面助手

![YuToys Badge](https://img.shields.io/badge/YuToys-v1.0.0-blueviolet?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Stable-success?style=for-the-badge)

**YuToys** 是一款专为追求极致视觉体验的业余爱好者打造的桌面辅助工具。它融合了番茄钟、任务管理、饮水提醒与美学统计，旨在成为您指尖最解压的数字艺术品。

---

## ✨ 核心特性

- 🎨 **极致美学**：采用 Glassmorphism（玻璃拟态）设计，支持透明磨砂质感与沉浸式动效。
- 🔔 **品牌化通知**：支持整点报时、饮水提醒，所有音效均已本地化，支持 100% 离线运行。
- 🚀 **性能卓越**：采用 Electron + Vite 架构，经过 v16 版本的极致启动优化（无压缩秒开技术），实现秒级响应。
- 💾 **数据持久化**：自动保存任务清单、累计饮水量及窗口位置信息。

## 🛠️ 技术栈

- **前端**: React 19 + TypeScript
- **状态管理**: Zustand
- **动效**: Framer Motion + Canvas Confetti
- **核心框架**: Electron 40
- **构建工具**: Vite + Electron Builder

## 📦 如何运行

如果您是开发者，可以尝试自行构建：

1. **安装依赖**
   ```bash
   npm install
   ```
2. **开发模式运行**
   ```bash
   npm run dev
   ```
3. **分发打包**
   ```bash
   npm run build
   ```
